# Random Wallpapers Collection 🎨

[![GitHub Repo Size](https://img.shields.io/github/repo-size/Slashdog29/wallparpers-ramdon?style=flat-for-the-badge)](https://github.com/Slashdog29/wallparpers-ramdon)
[![License](https://img.shields.io/badge/License-MIT-lightgrey?style=flat-for-the-badge)](https://opensource.org/licenses/MIT)

Esta es una colección curada de fondos de pantalla (wallpapers) de alta calidad, seleccionados para ofrecer una estética limpia y moderna a tu escritorio. El repositorio incluye imágenes elegidas cuidadosamente para mejorar tu entorno de trabajo, además de ser compatible con scripts de automatización para rotación aleatoria.

## ✨ ¿Por qué este repositorio?
Sabemos que un buen entorno visual ayuda a mantener la concentración. Esta colección está optimizada para:
* **Calidad visual:** Imágenes de alta resolución.
* **Estética:** Selección basada en estilos minimalistas y modernos.
* **Compatibilidad:** Ideales para entornos de escritorio Linux (Hyprland, Niri, GNOME, KDE).

## 🚀 Uso sugerido (Automatización)
Si utilizas **Hyprland** o cualquier otro gestor de ventanas, puedes automatizar el cambio de fondo usando un script simple con herramientas como `swww` o `hyprpaper`.

Ejemplo rápido para Linux:
```bash
# Script básico para rotar fondos aleatoriamente
feh --randomize --bg-fill /ruta/a/tu/repositorio/wallparpers-ramdon/*
